Tool Description
Selenium Open-source framework for automating web applications across different browsers and platforms.
Appium Open-source tool for automating mobile applications on various platforms (Android and iOS).
JUnit Java-based framework for writing and running unit tests and can be used for automation testing.
TestNG Test framework inspired by JUnit and TestNG, designed for test configuration and parallel execution.
Cucumber Behavior-driven development (BDD) tool that allows tests to be written in plain language.
Jenkins Open-source automation server that can be used to set up continuous integration and automation jobs.
Robot Framework Generic test automation framework for acceptance testing and acceptance test-driven development (ATDD).
Postman API testing tool for sending HTTP requests and testing responses, widely used for API automation.
TestComplete Commercial testing tool for automating desktop, web, and mobile applications.
